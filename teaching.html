Stanford

ACM 109. Mathematical Modelling. 2020 Spring

Instructor: Lexing Ying

Course Description: Numerical computation for engineering and machine learning applications: error analysis, floating-point arithmetic, numerical solution of linear and nonlinear equations, optimization, gradient descent, polynomial interpolation, numerical differentiation and integration, supervised learning, numerical solution of ordinary differential equations, numerical stability, unsupervised learning, sampling (Monte Carlo algorithms). Implementation of numerical methods in programming assignments (Python or Matlab).

Caltech

CS/CMS/CNS/EE/IDS 155: Machine Learning and Data Mining (Graduate level). 2022 Winter

Instructor: Yisong Yue

Course Description: This course will cover popular methods in machine learning and data mining, with an emphasis on developing a working understanding of how to apply these methods in practice. This course will also cover core foundational concepts underpinning and motivating modern machine learning and data mining approaches. This course will also cover some recent research developments.
